Sudoko Game

React JavaScript Vite Tailwind css

Project Overview

The Sudoku Game will allow users to play 9x9 Sudoku puzzles. It will feature puzzle generation, input validation, hints, a timer, and the ability to check and solve puzzles. Users can select difficulty levels (easy, medium, hard), reset the board, and track their best times.

Live Link

Key Features

  • Puzzle Generation: Randomly generate solvable Sudoku boards.
  • User Input: Allow users to fill in numbers with mouse/touch/keyboard.
  • Validation: Real-time or button-triggered checking of the board for mistakes.
  • Hints: Option to reveal a correct number.
  • Timer: Track how long the user takes to solve the puzzle.
  • Difficulty Levels: Easy, Medium, Hard puzzles.
  • Reset/Undo: Clear the board or undo the last move.
  • Save/Load Game: (Optional) Allow users to save progress.
  • Leaderboard: (Optional) Display best completion times.

Technical Implementation

The system implements various frontend techniques:

  • Frontend: HTML, Tailwind CSS, JavaScript (or React.js/vite.js for more dynamic UI)

Results

The system achieved over 85% accuracy in predicting user ratings and providing relevant interesting levels. The hybrid approach significantly improved level to level according to user interactive to the game.
